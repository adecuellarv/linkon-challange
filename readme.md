# Sistema de Reservaciones de Hote LinkON

Aplicación de consola desarrollada en **TypeScript** para gestionar las reservaciones de un hotel.


---

## Tecnologías

- Node.js
- TypeScript
- Inquirer
- ts-node

---

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/adecuellarv/linkon-challange
cd linkon-challange
```

---

## Instalación de dependencias

`npm install`


## Correr el proyecto

`npm run start`
